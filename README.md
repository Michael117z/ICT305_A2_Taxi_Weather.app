# ICT305 Taxi Operations Dashboard

Interactive decision-support dashboard developed using Streamlit.

Dataset:

- Taxi Availability
- Regional Taxi Summary
- Rainfall

Developed for ICT305 Assignment 2 Group Project.

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://ict305-a2-taxidashboard.streamlit.app/)

### How to run it on your own machine

Prerequisite: install `uv` if you don't already have it.

```
$ curl -LsSf https://astral.sh/uv/install.sh | sh
```

1. Clone the repository
git clone <repository-url>
cd ICT305_A2_Taxi_Weather.app

2. Install dependencies
pip install -r requirements.txt

or

uv sync

3. Start Streamlit
streamlit run streamlit_app.py

or

uv run streamlit run streamlit_app.py

The dashboard will open automatically in your web browser.
